Class PinView

    public final class PinView
extends PasscodeView
implements InteractiveArrayList.ChangeListener
    Created by Keval on 06-Apr-17.

    This view will perform the PIN based authentication. This view also support fingerprint authentication. To set this view application has to

  • 1. Set the correct PIN using setCorrectPin(int[]).
  • 2. Set key shape using setKey(Key.Builder).
  • 3. Set the callback listener. PasscodeView.setAuthenticationListener(AuthenticationListener)

    • This view is made up of three different views.
  • Title with the PIN indicators. BoxTitleIndicator
  • Keyboard. BoxKeypad
  • Fingerprint authentication view. BoxFingerprint
